Stanice pro radiovou detekci meteorů

[InfoLongDescription.en]

MLAB set of modules which are used for radio detection of meteor trails. Upgraded version with globaly synchronized local oscillator and improved digitalizing system. This version using an FPGA module to compensate local oscillator drifts.
